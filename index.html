Información general sobre el OVI

En el siguiente sitio encontrara información la cual esperamos sea de utilidad y le permita comprender la importancia de la seguridad informática para una empresa y para las personas.

Objetivos Del OVI

  • Dar a conocer en consiste la seguridad informática y la importancia de tener un buen mecanismo de seguridad.
  • Mostar cuales son las principales áreas que cubre la seguridad informática.
  • Enseñar los tipos de virus y su forma de ataque.
  • Concientizar sobre la importancia de contar con un buen mecanismo de seguridad tanto de prevención como de respuesta en caso de un ataque.

Objetivos del curso

  1. Aprender a crear de forma correcta el Repositorio con GITHUB.
  2. Aprender a publicar de forma correcta el repositorio en GITHUB PAGES.
  3. Aprender a crear de forma correcta el OVI con HTML5 y CSS3
  4. Aprender a crear de forma correcta las secciones y los contenidos solicitados para el OVI.
  5. Aprender a configurar de forma correcta el Contador de visitas para el sitio web del OVI.
  6. Aprender a configurar de forma correcta el Modulo de red social TWITTER UNAD para el OVI.

Temáticas

HTML5 y CSS3, para ser visualizados en diferentes navegadores teniendo en cuenta su funcionabilidad y usabilidad, por medio del uso de diferentes editores de código propuestos en el desarrollo del curso
